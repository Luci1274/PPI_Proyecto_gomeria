# Actores: Adminitrador

# Caminos Basico:
1. El administrador registra un producto
2. El sistema guarda el producto ingresado
3. El administrador consulta la lista de productos
4. El sistema muestra los productos en stock y los faltantes
5. El administrador ingresa los precios de los productos
6. El sistema actualiza los precios
7. El administrador descuenta el producto vendido
8. El sistema actualiza la cantidad de stock
9. El administrador elimina los productos que no vende mas
10. El sistema actualiza la lista de productos
